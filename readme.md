Nomad Jump Website
This repository hosts the front-end code for Nomad Jump, a lifestyle brand focused on combining jump rope fitness, travel, and adventure. Our mission is to provide the ultimate minimalist fitness solution for people who want to stay active anywhere in the world.
Brand Identity
• Primary Color: #E8761F (Orange)
• Secondary Color: #102E4A (Dark Blue)
• Core Philosophy: Your Gym in a Handle.
Project Structure
The website is a Single-Page Application (SPA) hosted on GitHub Pages, utilizing smooth scrolling to navigate between sections.
• index.html: The complete website, including all HTML structure, design (via Tailwind CSS), and content.
Key Sections
The website is structured to tell the Nomad Jump story and promote the ropes:
1. Hero: The brand mission statement and primary call-to-action (CTA).
2. Mission: Explains the core values: Freedom, Simplicity, Adventure, and Mindset.
3. Shop: Dedicated e-commerce section for the three signature ropes (Velocity, FlowMaster, Endurance).
4. Training: Links to YouTube tutorials and lists of training programs.
5. Blog: Showcases travel stories, fitness tips, and community events.
6. Contact: Provides easy access to email and social media.
Setup and Deployment
This website is designed to be instantly deployable via GitHub Pages.
1. Clone this repository.
2. Go to your GitHub repository settings.
3. Navigate to the "Pages" section.
4. Set the source branch to main and the folder to / (root).
5. Save the settings. Your website will be live at the provided GitHub Pages URL shortly!
Next Steps
Future development will focus on integrating with an external e-commerce platform (like Shopify) to handle secure payments, inventory, and logistics for the physical products.
